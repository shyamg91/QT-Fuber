# QT-Fuber

To get started
``` 
Clone the repo, git clone https://github.com/shyamg91/QT-Fuber.git
```

Once inside the folder 
1. To run the app,
```
npm run watch
```
2. To run the tests,
```
npm test
```

API request format
```
http://localhost:7777/driver?lat=-36.889528&lng=174.693118&type=Pink
```
